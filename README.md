simpleaws
=========

An idiot-proof way to make a public AWS based service.

# Installation

1. Install it.

    ```bash
    pip install simpleaws
    ```

2. Copy settings\_template.py to settings.py and enter your keys.

3. Run test\_simpleaws.py.

    ```bash
    python test_simplaws.py
    ```
